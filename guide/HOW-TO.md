# BOH Shift Ops — Leader Guide (new system)

**Chick-fil-A Vestavia Hills · Back of House**

This is the **current** product — not the old open shared board.

- Everyone **signs in**
- Managers build Schedule / Leaders / Shift Setup and **Publish setup**
- Leads & teammates use **My Week** (hours + leadership + stations)
- Admin controls users and sees unpublished dayparts

## What’s new vs the old system

| Old | New |
|-----|-----|
| Open page, anyone edits | Sign-in required |
| No personal view | **My Week** per person |
| Leadership only on Leaders/Setup | Leadership also on **My Week** |
| Board “done” when printed | **Publish setup** per period |
| No access control | Admin / Manager / Team member roles |
| App lived at site root | App is **`/app.html`** (root = landing) |

## Roles

| Role | Access |
|------|--------|
| **Admin** | All tabs + users + unpublished board + reset |
| **Manager** | My Week, Schedule, Setup, Skills, Leaders, Publish |
| **Team member** | My Week only |

Shift leaders who build the week need a **Manager** login.

---

## Manager week flow

1. **Sign in** → link roster name  
2. **Schedule** — Mon–Sat in/out times for the team  
3. **Leaders** — BF / LN / DN each day (~4:30a–11a / 10a–3p / 3p–11p)  
4. **Shift Setup** — place people on stations  
   - Phone: **double-tap** + Add person  
   - Desktop: drag from pool  
5. **Publish setup** for that period  
6. **Print** 2-page gameplan (BF+LN / Afternoon+Dinner)

**Golden rule:** Schedule first or the Setup pool is empty. Leaders stay in the Setup header, not the station pool.

---

## My Week (leads & team)

1. Link your roster name  
2. Open **My Week**  
3. See:
   - Lead-day chips (ex: `Mon · LN Lead`)
   - Leadership windows
   - Scheduled hours
   - Station positions (may say **draft** until published)

---

## Admin

1. **Admin** tab → **+ Add user** (manager or member)  
2. Set password + optional roster link  
3. Use **Unpublished setups** to find dayparts still missing Publish
