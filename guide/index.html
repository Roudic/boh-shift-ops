How to use (photo demos)

1 · Home

Open the app

Start on the live site. Top bar: Schedule / Shift Setup, week nav, cloud badge, Export.

  1. Open the Live app link above
  2. Use Schedule for the week grid
  3. Use Export to download a JSON backup
2 · Classic

Paint the week schedule

Click-drag empty cells, stretch edges, sort by start time with day arrows.

  1. Drag empty cell to create a shift
  2. Drag block to move · edges to stretch
  3. ▲ earliest first · ▼ latest first
3 · SmartView

Hour timelines

HotSchedules-style multi-day hours with labor footer.

  1. Click SmartView
  2. Scroll Mon–Sat hour rulers
  3. Sort with ▲ ▼ on day headers
  4. ClassicView to exit
4 · Setup

Periods, positions, handovers

Build your own period blocks and station boards. Stack people for 3pm handovers.

  1. Period Blocks → + Add period
  2. Shift Setup → Edit Positions
  3. Link layout to multiple periods
  4. + Add person / handover on same spot
5 · Cloud

Connect & share

Top bar cloud badge → paste Supabase URL + anon key → Save · Test · Push.

  1. Project URL: https://oikrcoccaepgtvdswuuo.supabase.co
  2. Paste your anon public key in the app only
  3. Push once · others Pull on their devices
  4. Share this guide page link with the team
