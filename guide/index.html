Step by step (current product)

1 · Sign in

Open the app and log in

The app is locked until you sign in. Ask Admin for a username and password.

  1. Go to https://roudic.github.io/boh-shift-ops/app.html
  2. Enter username + password → Sign in
  3. When asked, pick who you are on the roster (Link my roster name)
  4. Cloud should connect by itself — badge says Cloud
Product landing page is the root URL. The working tool is always /app.html.
Roles

What each login can see

Admin controls access. Your role decides which tabs appear.

  • Admin — every tab + Users + unpublished setups + Reset
  • Manager — My Week, Schedule, Shift Setup, Skills, Leaders, Publish
  • Team member — My Week only
Shift leaders who build the week need a Manager account. Floor team uses Team member.
2 · Schedule

Build the Mon–Sat week

Managers put every BOH person on the grid with real in/out times.

  1. Open Schedule
  2. Use week arrows for the week you’re building
  3. Click an empty day cell → set start/end → Save
  4. Use + Shift for a second block the same day
  5. Drag a shift card to move it to another person/day
  6. Optional: SmartView for hour timelines (paint / stretch bars)
Golden rule: Schedule first. If nobody is scheduled for a period, Shift Setup’s team pool is empty.
3 · Leaders

Assign Breakfast / Lunch / Dinner leads

Leads run the period. They show in Setup headers — and on their own My Week.

  1. Open Leaders
  2. Add leaders to the roster if needed
  3. Assign each day:
    • BF Lead ~ 4:30a–11:00a
    • LN Lead ~ 10:00a–3:00p
    • DN Lead ~ 3:00p–11:00p
  4. Leadership hours can auto-add on the Schedule
  5. On Shift Setup, leads appear as header chips (not in the station pool)
4 · Shift Setup + Publish

Set the floor, then publish the period

This replaces the old “just leave the board open” habit — publish when it’s ready.

  1. Open Shift Setup → pick Day + Period
  2. Confirm BF/LN/DN lead chips in the header
  3. Place people on stations from the team pool
    • Phone/tablet: double-tap + Add person → pick name → times
    • Desktop: drag name onto a station → times → Save
  4. Add handovers with + Add person / handover
  5. Fill checklist + notes for that period
  6. Click Publish setup — badge shows Published + your name
Until you publish, My Week may show station rows as draft. Admin sees unpublished dayparts on the Admin tab.
5 · My Week

Personal week — hours, leadership, stations

This is the new home screen for leads and team members.

  1. Link your roster name (once)
  2. Open My Week
  3. See week chips for lead days (example: Mon · LN Lead)
  4. Each day shows:
    • Leadership window if you’re BF/LN/DN that day
    • Scheduled hours
    • Station positions (draft until published)
If My Week is empty, you’re not linked to a roster person — or that person has no hours/leads this week.
6 · Admin

Users, access, unpublished setups

Admin only (Joshua / Roudic). Create manager and member logins here.

  1. Open Admin
  2. + Add user → display name, username, password, role
  3. Optionally link them to a roster person
  4. Check Unpublished setups — jump straight to any missing period
Print

Print the 2-page gameplan

Hang the board copy after the period is set (and preferably published).

  1. Shift Setup → correct day
  2. Print → landscape
  3. Page 1 = Breakfast + Lunch · Page 2 = Afternoon + Dinner
Export ZIP also builds those staffing pages plus a week schedule PDF.

Before rush checklist (new system)

  1. Signed in as Manager (or Admin)
  2. Cloud badge shows connected
  3. Correct day + period on Shift Setup
  4. Lead chips correct for BF / LN / DN
  5. Stations filled · handovers timed
  6. Publish setup clicked
  7. Print gameplan if the paper board is stale
  8. Leads can open My Week to confirm their leadership window
