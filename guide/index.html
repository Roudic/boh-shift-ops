Your job as shift leader — 4 steps

1

Build the week

Put every BOH person on the Schedule with real in/out times.

2

Name the leads

Set BF / LN / DN leaders under Leaders so hours land on the grid.

3

Set the floor

Open Shift Setup → drag people onto stations for that daypart.

4

Print & run

Print the 2-page gameplan (BF+LN / Afternoon+Dinner) for the board.

Golden rule

  1. Schedule first — if nobody is on the grid for that period, the Setup pool is empty.
  2. Then Setup — stations only use people already scheduled on that block.
  3. Leaders stay out of the pool — they show in the Setup header (BF / LN / DN chips).

How to use — step by step

1 · Schedule

Build the week schedule

This is where people get hours. Do this before station setup.

  1. Open the app → top tab Schedule.
  2. Use week arrows to pick Mon–Sat of the week you’re building.
  3. ClassicView (default): each row is a person, each column is a day.
    • Click an empty cell → set start / end → Save.
    • Or click + Add shift on a day that already has hours (multi-shift same day).
    • Drag a shift card to another person or day to move it (not copy).
    • Use Edit on the card to change times; ✕ removes that block.
  4. Day headers show how many people are scheduled + BF/LN/DN lead chips.
  5. When the week looks right, you’re ready for Shift Setup.
Leader tip: Paint coverage for Breakfast, Lunch, Afternoon, and Dinner windows so Setup pools are full when you assign stations.
2 · SmartView

Paint hours on the timeline

Use SmartView when you want HotSchedules-style hour bars.

  1. On Schedule, click SmartView (toolbar or a day header).
  2. Each row is a person; columns are days with hour marks (about 4:30a–11p).
  3. Drag on empty track to paint a new shift.
  4. Drag the bar to slide times; drag left/right edges to stretch start/end.
  5. Drag a bar onto another person’s row to reassign (it moves — no duplicate).
  6. ClassicView or Esc exits back to the week grid.
Pan the days: Shift + scroll, or drag from the name column, to move sideways while names stay sticky.
3 · Leaders

Assign BF / LN / DN leaders

Leaders run the period — they show on the Setup header, not in the station pool.

  1. Open the Leaders tab.
  2. Add leaders to the roster if needed.
  3. For each day, assign:
    • Breakfast Lead ~ 4:30a–11:00a
    • Lunch Lead ~ 10:00a–3:00p
    • Dinner Lead ~ 3:00p–11:00p
  4. Assigning a lead can auto-add leadership hours on the schedule if they weren’t already on.
  5. On Shift Setup, those names appear as chips under the title (gold outline when that lead overlaps the period you picked).
Don’t put the lead on a station unless you really need them on the board — the pool deliberately hides day leaders so floor spots stay for the team.
4 · Shift Setup

Set stations for a period

This is the floor board: who is on Primary, Machines, Breading, Fries, etc.

  1. Open Shift Setup.
  2. Pick Day and Period block (Breakfast, Lunch, Afternoon, Dinner…).
  3. Team pool (right/side) lists people scheduled on that block — sorted by out time for handoffs.
  4. Drag a name onto a station pad → confirm start/end times → Save.
  5. Or click a name (selects them), then click a station.
  6. For a 3p handover on the same station:
    • First person already on the pad
    • Click + Add person / handover
    • Second person + times (e.g. 5a–3p then 3p–close)
  7. Click the ✕ on a person row to pull them off that station (back to pool).
  8. Use each period’s own checklist on the side — not shared across periods.
Empty pool? Nobody is scheduled for that period yet — go back to Schedule and add hours that overlap the period times.
5 · Print

Print the BOH gameplan (2 pages)

What you hang for the team — centerline stations + side roster.

  1. Still on Shift Setup, pick the day you’re running.
  2. Click Print.
  3. You get exactly two landscape pages when both halves of the day exist:
    • Page 1 — Breakfast & Lunch
      Station boards for early dayparts + combined side roster for those people
    • Page 2 — Afternoon & Dinner
      Station boards for late dayparts + combined side roster
  4. Print or Save as PDF from the browser dialog.
ZIP export (top bar Export) uses the same 2-page staffing layout per day, plus a week schedule PDF.

Before rush — 2-minute check

Leader pre-shift checklist

  1. Correct day selected on Shift Setup.
  2. Period matches the rush you’re prepping (Breakfast / Lunch / Afternoon / Dinner).
  3. Lead chip in the header is the right person for that window.
  4. Primary / Machines / Breading / Fries have names (or a clear open slot you own).
  5. Handovers timed (who leaves, who takes over, what time).
  6. Period checklist scanned; notes box has any callouts.
  7. Print gameplan if the board copy is stale.

Cloud (short version for leaders)

Cloud

Stay in sync with the store

You don’t need to set up Supabase every time — only pull/push once the store is connected.

  1. Top bar cloud badge → if it says connected, click Pull from DB when you open the app on a new browser.
  2. After big schedule/setup changes, Push to DB so the next leader gets your work.
  3. If you’re not on cloud, use Export (ZIP or JSON) and share the file.
Last push wins — pull before a long edit session if someone else may have updated the week.
