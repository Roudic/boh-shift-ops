Before you build — sign in

Accounts & access

  • Admin — full control, users & roles, unpublished setup board (Joshua / Roudic).
  • Manager — Schedule, Shift Setup, Skills, Leaders, Publish.
  • Team member — My Week only (hours, leadership, stations).
  1. Open the app → enter the username/password your admin gave you.
  2. Choose who you are on the roster (Link my roster name) so My Week works.
  3. Cloud connects automatically on the live site — badge should say Cloud, not Local only.

Your job as shift leader — 5 steps

1

Build the week

Put every BOH person on the Schedule with real in/out times.

2

Name the leads

Set BF / LN / DN leaders under Leaders so hours land on the grid.

3

Set the floor

Open Shift Setup → place people on stations for that daypart.

4

Publish

Tap Publish setup when that period board is ready for the team.

5

Print & run

Print the 2-page gameplan for the board; leads check My Week.

Golden rule

  1. Schedule first — if nobody is on the grid for that period, the Setup pool is empty.
  2. Then Setup — stations only use people already scheduled on that block.
  3. Leaders stay out of the pool — they show in the Setup header (BF / LN / DN chips).
  4. Publish when ready — Admin can see which dayparts still need publish.

How to use — step by step

1 · Schedule

Build the week schedule

This is where people get hours. Do this before station setup.

  1. Sign in → top tab Schedule (managers/admins).
  2. Use week arrows to pick Mon–Sat of the week you’re building.
  3. ClassicView (default): each row is a person, each column is a day.
    • Click an empty cell → set start / end → Save.
    • Or click + Add shift on a day that already has hours.
    • Drag a shift card to another person or day to move it (not copy).
    • Use Edit on the card to change times; ✕ removes that block.
  4. Day headers show headcount + BF/LN/DN lead chips.
  5. When the week looks right, go to Leaders / Shift Setup.
Leader tip: Paint coverage for Breakfast, Lunch, Afternoon, and Dinner windows so Setup pools are full when you assign stations.
2 · SmartView

Paint hours on the timeline

Use SmartView when you want HotSchedules-style hour bars.

  1. On Schedule, click SmartView (toolbar or a day header).
  2. Each row is a person; columns are days with hour marks (about 4:30a–11p).
  3. Drag on empty track to paint a new shift.
  4. Drag the bar to slide times; drag left/right edges to stretch start/end.
  5. Drag a bar onto another person’s row to reassign (it moves).
  6. ClassicView or Esc exits back to the week grid.
Pan the days: Shift + scroll, or drag from the name column, to move sideways while names stay sticky.
3 · Leaders

Assign BF / LN / DN leaders

Leaders run the period — they show on the Setup header, not in the station pool. Their My Week also shows leadership windows.

  1. Open the Leaders tab.
  2. Add leaders to the roster if needed (+ Add leader).
  3. For each day, assign:
    • Breakfast Lead ~ 4:30a–11:00a
    • Lunch Lead ~ 10:00a–3:00p
    • Dinner Lead ~ 3:00p–11:00p
  4. Assigning a lead can auto-add leadership hours on the schedule.
  5. On Shift Setup, those names appear as chips under the title (highlighted when that lead overlaps the period you picked).
  6. On My Week, a linked lead sees chips like Mon · BF Lead plus the leadership time window for each day.
Don’t put the lead on a station unless you really need them on the board — the pool hides day leaders so floor spots stay for the team.
4 · Shift Setup + Publish

Set stations, then publish the period

This is the floor board: who is on Primary, Machines, Breading, Fries, etc.

  1. Open Shift Setup.
  2. Pick Day and Period block (Breakfast, Lunch, Afternoon, Dinner…).
  3. Confirm the lead chip in the header for that window.
  4. Team pool lists people scheduled on that block (sorted by out time).
  5. Phone / tablet: double-tap + Add person on a station → pick from the pool → set times.
  6. Desktop: drag a name onto a station pad → set from/until → Save (or click name, then station).
  7. For a handover on the same station: + Add person / handover → second name + times.
  8. Fill that period’s checklist and notes.
  9. When the board is ready, click Publish setup. Badge flips to Published (with your name).
Empty pool? Nobody is scheduled for that period yet — go back to Schedule first.
Admin: unpublished dayparts show on the Admin board until you publish them.
5 · My Week

Personal week for every teammate (and every lead)

Anyone signed in can open My Week after linking their roster name.

  1. Use Link my roster name (or the prompt after first login).
  2. Open My Week.
  3. You’ll see scheduled hours, leadership (BF / LN / DN) when you’re a lead that day, and posted station positions.
  4. Draft station rows show as draft until that period is Published.
Team members only see My Week. Managers/admins see every tab — still use My Week to check your own lead days.
6 · Print

Print the BOH gameplan (2 pages)

What you hang for the team — centerline stations + side roster.

  1. Still on Shift Setup, pick the day you’re running.
  2. Click Print.
  3. You get exactly two landscape pages when both halves of the day exist:
    • Page 1 — Breakfast & Lunch · boards + combined side roster
    • Page 2 — Afternoon & Dinner · boards + combined side roster
  4. Print or Save as PDF from the browser dialog.
ZIP export (More / Export ZIP) uses the same 2-page staffing layout per day, plus a week schedule PDF.

Before rush — 2-minute check

Leader pre-shift checklist

  1. You’re signed in and cloud shows connected (if using the live app).
  2. Correct day selected on Shift Setup.
  3. Period matches the rush you’re prepping (Breakfast / Lunch / Afternoon / Dinner).
  4. Lead chip in the header is the right person for that window.
  5. Primary / Machines / Breading / Fries have names (or a clear open slot you own).
  6. Handovers timed (who leaves, who takes over, what time).
  7. Period checklist scanned; notes box has any callouts.
  8. Publish setup clicked for this period.
  9. Print gameplan if the board copy is stale.

Cloud (short version for leaders)

Cloud

Stay in sync with the store

The live app auto-connects to the Vestavia store cloud on every phone/PC.

  1. Open https://roudic.github.io/boh-shift-ops/app.html → sign in.
  2. Badge should say Cloud (not Local only). Refresh once if it doesn’t.
  3. Edits auto-push while connected · use Pull if someone else just changed the board.
  4. Last push wins — pull before a long edit session if others are working.
  5. Backup option: Export ZIP or Backup JSON.
Product landing (share with outsiders): https://roudic.github.io/boh-shift-ops/ · team always bookmarks the app link above.
