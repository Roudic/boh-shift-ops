Pre-rush checklist

0/9

    Shift notes

    New Auto-Schedule Template

    Times are read from Shift Periods (sidebar). Here you only set how many people each block needs.

    Add period block

    Presets only copy a starting board — every position is fully editable after.

    Edit Positions

    Build groups & positions once, link to multiple periods

    Build your station board below, then check every period block that should use it. One save applies to all selected periods so you don’t redo the same groups over and over. Optional times are default cover windows — handovers still work on the board.

    Check Breakfast + Mid + Closing (etc.) to share one board.

    Assign position

    Same position can hold multiple people with different times for handovers (e.g. ends 3:00p, next starts 3:00p).

    Add Team Member

    Copy shift to employees

    Existing shifts on selected people/days are replaced. Outside-availability warnings are skipped so you can schedule freely — check the grid for ⚠ flags after.

    Import HotSchedules Schedule

    Imports published shifts from HotSchedules export format: Name, Day, Start, End, Hours. Team members are matched by name; new names are added to roster.

    Import Roster & Availabilities

    One import adds your whole team and their availability times (HotSchedules CSV/Excel, staff exports, JSON). Replaces the current team and loads the week from your file.
