    Edit Positions

    Per daypart board · sticks every day & week

    Set the station board for each daypart (Breakfast, Lunch, Afternoon, Dinner…). Once saved, that daypart’s positions stay the same on every day and every week — only who you put on the pads changes. Unchecked dayparts are left alone (so Lunch won’t overwrite Breakfast).

    Tip: edit Breakfast → check only Breakfast → Save. Then do Lunch separately. Each keeps its own board forever.
